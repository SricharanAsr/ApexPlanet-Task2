# ApexPlanet-Task2: EDA & Business Intelligence

## 🎯 Objective
This project focuses on performing Exploratory Data Analysis (EDA) on synthetic sales data to uncover significant trends, customer behaviors, and actionable business insights. Using Python for advanced statistical analysis and SQL for database querying, we aim to provide a data-driven approach to business decision-making within the retail domain.

## 📂 Project Structure
- `eda_report.md`: Final summary of findings and business recommendations.
- `dashboard_mockup.md`: Strategic KPI dashboard design.
- `business_queries.sql`: Advanced SQL queries for revenue and demographic analysis.
- `eda_analysis.py`: Python automation for statistical analysis and visualization.
- `plots/`: Visual evidence of data trends (distribution, correlation, categorical analysis).
- `sales_data.csv`: Synthetic dataset generated for the analysis.
- `sales_database.db`: SQLite database for SQL query verification.

## 🛠 Tech Stack
- **Language**: Python 3.x
- **Libraries**: Pandas, Matplotlib, Seaborn, NumPy
- **Database**: SQLite
- **Tools**: GitHub, SQL

## 📈 Key Metrics Explored
- Total Revenue & Average Transaction Value
- Monthly Sales Trends
- Customer Demographics (Age Group Analysis)
- Category Popularity & Performance
